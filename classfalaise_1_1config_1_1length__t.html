Detailed Description

quantity for values with dimension tag length ([L])

A length_t value may be constructed using, for example

falaise::config::length_t x{3.14, "TAG"};

where TAG is one of the following valid unit tags:

  • angstrom
  • astronomical_unit
  • centimeter
  • decameter
  • decimeter
  • femtometer
  • foot
  • gigalight_year
  • gigaparsec
  • hectometer
  • inch
  • kilolight_year
  • kilometer
  • kiloparsec
  • light_year
  • megalight_year
  • megaparsec
  • meter
  • micrometer
  • mile
  • millimeter
  • nanometer
  • nautical_mile
  • parsec
  • picometer
  • yard

A wrong_dimension_error will be thrown if the supplied unit tag is not in the above list
