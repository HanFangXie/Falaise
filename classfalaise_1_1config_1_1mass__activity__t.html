Detailed Description

quantity for values with dimension tag mass_activity ([M-1][T-1])

A mass_activity_t value may be constructed using, for example

falaise::config::mass_activity_t x{3.14, "TAG"};

where TAG is one of the following valid unit tags:

  • Bq/kg
  • GBq/kg
  • MBq/kg
  • dpm/kg
  • kBq/kg
  • mBq/kg
  • uBq/kg

A wrong_dimension_error will be thrown if the supplied unit tag is not in the above list
