§ data_description_mask

enum snemo::datamodel::raw_calorimeter_hit::data_description_mask

Bit masks to describe data.

Bit contents: [UUUU UUUU UUUT DCBW]

Enumerator
DATA_DESC_UNDEFINED 

The undefined data description.

DATA_DESC_SIG_HASWAVEFORM 

The raw data stores the waveform [W].

DATA_DESC_AUX_HASBASELINE 

The raw data stores the computed baseline [B].

DATA_DESC_AUX_HASPEAKCHARGE 

The raw data stores the computed peack charge [C].

DATA_DESC_AUX_HASPEAKDATA 

The raw data stores other values related to the peak [D].

DATA_DESC_AUX_HASTIMECFDINTERP 

The raw data stores the interpolated CFD timestamp [T].

DATA_DESC_UNUSED 

Unused bits [U].
