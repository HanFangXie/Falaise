
EventBrowser is a C++ program library plugin for Falaise,
the SuperNEMO experiment simulation and data analysis
software suite.

flsimulate is a C++ program library which allow to display
Monte Carlo and real events.

flsimulate depends on the Falaise and Bayeux libraries.
